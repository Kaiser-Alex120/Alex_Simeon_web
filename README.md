# 🌐 Portafolio Web – Alex Simeón

Bienvenido a mi espacio digital, donde combino economía, análisis de datos y tecnología para construir soluciones prácticas y visuales.

## 🚀 Sobre el proyecto

Este sitio web presenta una colección de herramientas y proyectos enfocados en la **extracción, procesamiento y visualización de datos económicos y financieros**. El objetivo es transformar datos complejos en información clara y útil para la toma de decisiones.

## 📊 Principales funcionalidades

- 🔎 Extracción de datos desde fuentes como:
  - Yahoo Finance  
  - Banco Mundial  

- 📈 Visualización de datos económicos y financieros  
- 🧠 Aplicación de herramientas analíticas y econométricas  
- 💻 Integración con proyectos desarrollados en GitHub  

## 🛠️ Tecnologías utilizadas

- Python (análisis de datos)
- APIs financieras y económicas  
- HTML, CSS y JavaScript  
- GitHub Pages para despliegue  

## 🎯 Enfoque

Mi trabajo se centra en:

- Data-driven decision making  
- Automatización de procesos de análisis  
- Visualización clara y profesional  
- Integración entre economía y programación
