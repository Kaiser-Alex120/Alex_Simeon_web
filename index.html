VarPy

Python VAR / SVAR Econometría APIs

Pipeline completo para modelos VAR/SVAR con restricciones de signo. Conexión directa a APIs del BCRP y Banco Mundial. Incluye: pruebas de estacionariedad (ADF, PP, KPSS), estimación VAR automática, identificación SVAR estructural, funciones impulso-respuesta, descomposición de varianza (FEVD), diagnósticos completos y pronósticos con bandas de confianza bootstrap. Totalmente modularizado y documentado.
