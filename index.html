VarPy

Python VAR / SVAR Econometrics APIs

Complete pipeline for VAR/SVAR models with sign restrictions. Direct connection to BCRP and World Bank APIs. Includes: stationarity tests (ADF, PP, KPSS), automatic VAR estimation, structural SVAR identification, impulse-response functions, variance decomposition (FEVD), complete diagnostics and forecasts with bootstrap confidence bands. Fully modularized and documented.
